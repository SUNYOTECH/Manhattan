## Nebula回放客户端

基于Html5实现

### 项目结构

- Server 存放与服务器交互的后台代码

- Client 前端页面

    - js 存放javascript等脚本

    - css 存放css样式表

    - img 存放项目相关的图片

- Docs 项目开发文档， 统一采用markdown语法编写. 每开发一个新项目需要更新相应的文档

- Test 测试目录，用于日常开发的测试所需.

    - js

    - img

    - css

    - html html文件存放在此

- index.html 入口文件，也是主要的显示界面, 就目前来说应该就这个一个html文件.
